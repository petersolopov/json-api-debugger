### 0.0.2

New Features:

- added enable/disable logging in options
- added http method in the log:
```
▶ GET /jsapi/entity
▶ POST /jsapi/entity 
```
- added the request body  For methods POST, PUT, PATCH e.g:
```
▼ POST /jsapi/entity/
  request: ▶ Object
  response: ▶ Object
  ...
```

Misc:

- renamed *deserialized* to *response*
